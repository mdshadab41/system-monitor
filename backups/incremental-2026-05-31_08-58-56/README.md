# System Monitoring & Automation Suite
Bash-based monitoring solution for Linux servers deployed on AWS.

## Tech Stack
Bash · Linux · Cron · rsync · Git · AWS EC2 · AWS S3

## Features (in progress)
- CPU, Memory, Disk, Network monitoring
- Automated log rotation
- Full & incremental backups to S3
- Interactive terminal dashboard
- Multi-format reports (HTML, CSV, Text)
- 24/7 cron scheduling
